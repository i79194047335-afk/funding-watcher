# Funding Watcher

Наблюдатель за ставками фандинга на Hyperliquid.
Тянет данные по перпам и подсвечивает интересные перекосы для market-neutral стратегии.

**Статус:** работает 24/7 на VPS по cron. Все 6 шагов + ревью закрыты (июнь 2026).

## Возможности

- Мониторинг ~230 перпов: фандинг, объём 24ч, наличие спота
- Три группы кандидатов: ✅ доступны (≥0.10%/ч) / 👀 наблюдаем / ✗ пропускаем
- Уведомления в Telegram на вход (новый кандидат) и на выход (пора закрывать)
- Только положительный фандинг (отрицательные недоступны без шорта спота)
- История в CSV (230 строк/запуск) — видно тренд, а не только момент
- Логирование ошибок API и событий
- Защита от спама: не дублирует уведомления

## Стек
- Python 3.10
- requests, python-dotenv

## Запуск
Разово (вручную):

    python3 watcher.py

24/7 через cron (каждые 15 минут):

    */15 * * * * cd /root/projects/funding-watcher && /usr/bin/python3 watcher.py >> logs/watcher.log 2>&1

## Настройки
- `.env` — токен Telegram-бота и chat_id (не в git)
- Пороги в начале `watcher.py`: `MIN_FUNDING_PCT=0.05`, `MIN_ACTIONABLE_PCT=0.10`, `MIN_VOLUME_USD=1_000_000`
- Уведомления только при фандинге ≥ `MIN_ACTIONABLE_PCT` (отбивает комиссии)
