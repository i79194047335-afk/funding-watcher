## Что это за проект

Бот-наблюдатель за ставками фандинга (funding rate) на бирже Hyperliquid.
Смотрит перпы (бессрочные контракты) по всем монетам и подсвечивает те, где
перекос ставки большой — это кандидаты на market-neutral стратегию
(спот + противоположный шорт/лонг, чтобы собирать фандинг независимо от цены).

**Важно про деньги:** реальная цель не доход (на бюджете ~$200 это центы),
а навык + рабочий бот для портфолио/фриланса. На старте бот ничем не рискует —
он только смотрит и считает. Реальные сделки — позже и крошечные.

## Стратегия в одну фразу

Фандинг капает каждый час. `+` ставка → лонги платят шортам (собираем, будучи в шорте).
`−` → шорты платят лонгам (собираем в лонге). Порог, при котором сделка вообще
имеет смысл (отбить комиссии): **~0.1% в час**. На мейджорах край сжат, живёт
длинный хвост (мелкие/свежие монеты). Гонка ботов (MEV/Solana/мосты/стейблы) —
не наш профиль, бюджет и латентность не те.

## Технические детали

- Python 3.10, библиотеки `requests`, `pandas`, `numpy` уже стоят в системе.
- API Hyperliquid: `POST https://api.hyperliquid.xyz/info`, без ключей и регистрации.
  Запрос фандинга: `{"type": "metaAndAssetCtxs"}` → `data[0]["universe"]` (имена монет),
  `data[1]` (контексты, поле `ctx["funding"]` — ставка за час в долях).
- Запуск: `python3 watcher.py` из `/root/projects/funding-watcher`.
- Папки `logs/` и `data/` существуют, но в git не попадают.
